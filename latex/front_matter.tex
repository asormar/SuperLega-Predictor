% RESUMEN (ES)
%=============================================================================
\pagenumbering{roman}
\resumen{Desarrollo de un modelo predictivo basado en Machine Learning para
         la estimaci\'on de resultados y an\'alisis de rendimiento en el
         voleibol profesional.}
{%
Este Trabajo de Fin de Grado presenta \textsc{Predictor(2)}, un sistema
completo para simular partidos y temporadas de la SuperLega italiana de
voleibol. La arquitectura combina un motor de \emph{cadenas de Markov}
punto a punto con tres modelos de \emph{\ac{ML}} (regresi\'on
log\'istica, gradient boosting y ensembles) entrenados sobre datos p\'ublicos
de los \'ultimos diez a\~nos. El sistema se complementa con una interfaz
web en React y una API REST en FastAPI. La evaluaci\'on con protocolo
\emph{rolling-origin} honesto (sin leakage temporal) arroja un AUC de
\num{0.762} para la predicci\'on de partido y \num{0.697} para la
predicci\'on de set (test 2025; media CV 2-fold \num{0.679}), con tiempos de simulaci\'on de temporada completa del
orden de \qty{70}{\second}.
}

% ABSTRACT (EN)
%=============================================================================
\resumenEN{Development of a Machine Learning-based predictive model for
           outcome estimation and performance analysis in professional
           volleyball.}
{%
This Bachelor's Thesis presents \textsc{Predictor(2)}, a complete system
for simulating matches and seasons of the Italian SuperLega volleyball
league. The architecture combines a point-by-point \emph{Markov chain}
engine with three \emph{machine learning} models (logistic regression,
gradient boosting, and ensembles) trained on publicly available data from
the last ten years. The system is complemented by a React web interface
and a FastAPI REST API. Evaluation with an honest \emph{rolling-origin}
protocol (no temporal leakage) yields an AUC of \num{0.762} for match
prediction and \num{0.697} for set prediction (test 2025; mean CV 2-fold \num{0.679}), with full-season
simulation times in the order of \qty{70}{\second}.
}
